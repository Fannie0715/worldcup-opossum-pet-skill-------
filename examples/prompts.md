# Example Prompts

## 一键安装默认负鼠

```text
Use $worldcup-opossum-pet 安装世界杯负鼠宠物。
保留默认负鼠主体，显示中国时间当天赛程，状态文案用熬夜看球但还要上班的语气。
```

## 改成我的主队版本

```text
Use $worldcup-opossum-pet 把负鼠改成阿根廷主队版。
要求：
- 负鼠本体不要变
- idle/running/review 都用阿根廷风
- waiting/waving 用巴西风保留亮色，并使用完整耳朵黄色参考图
- failed 用秘鲁风
- 赛程牌放脚边
- 文案更冷幽默一点
```

## 只改状态文案

```text
Use $worldcup-opossum-pet 只改状态文案，不改球衣和赛程牌。
语气：熬夜看球、白天装作在上班、冷静但崩溃。
```

## 只生成配置，不安装

```text
Use $worldcup-opossum-pet 先不要安装。
根据我的偏好生成一份 pet.config.json：
- 宠物名：加时赛负鼠
- 主队：法国
- 显示全天中国时间赛程
- 体积 normal
- 清晰度 extra-crisp
```

## 做 Codex 宠物展示图

```text
Use $worldcup-opossum-pet 为当前宠物生成一张状态对应图。
需要展示 9 个 Codex 状态、对应球队风格、状态文案和赛程牌。
用途：GitHub README 或 Codex 宠物说明页。
```
